# Penn State IST597 HCAI 2026 Spring: In-Class Material

## Overview

This is a collection of Streamlit-based chatbot projects, including multiple activities and exercise files designed to help users learn and implement different types of chat agents.

## File Structure

Below is an overview of the project's main files and directories:

- `activity-1-1.py`: A simple example of a Streamlit application.
- `activity-1-2.py`: Implements a basic chat input box.
- `activity-1-3.py`: Contains a chat agent that echoes user input.
- `activity-1-4.py`: Extends the chat agent to support chat history.
- `activity-2-1.py`: A rule-based chat agent.
- `activity-2-2.py`: A chat agent using the OpenAI API.
- `assignment-2-exercise.py`: An exercise file for a daily anxiety tracker.
